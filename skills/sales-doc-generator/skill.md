# 営業資料生成スキル

## 概要

営業用の資料（提案書、メール、フォローアップ）を効率的に生成するためのスキルです。

## 使い方

このスキルは以下の手順で実行します：

1. **顧客情報の入力**: 顧客の業界、規模、課題を入力
2. **資料種類の選択**: 提案書 / メール / フォローメール
3. **5体パイプライン実行**: 営業用途に最適化したパイプラインで生成

## 入力形式

```
顧客名: [会社名]
業界: [業界名]
規模: [従業員数]
課題: [具体的な課題]
資料種類: 提案書/メール/フォロー
```

## 出力先

- 提案書: `06_marketing/proposals/YYYY-MM-DD_企業名/proposal.md`
- メール: `06_marketing/email-templates/YYYY-MM-DD_用途.md`
- フォロー: `06_marketing/email-templates/YYYY-MM-DD_フォロー.md`
