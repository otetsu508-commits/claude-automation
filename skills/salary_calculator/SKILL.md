# Salary Calculator (給料計算スキル)

このスキルは、ユーザーのバイトの勤務時間をもとに給料を計算し、指定されたExcelファイルにその記録を残すためのものです。
「Excelなどの表計算ソフト」は、データを自動的にまとめるために使用されます。

**仕様:**
- **20日締め**: 毎月21日から翌月20日までの勤務が、該当月（21日以降の場合は翌月）の給料として記録されます。例えば、4月21日の勤務は「5月」のシートに記録されます。

## 使用方法 (AI向け)

ユーザーから「今日のバイトの給料を記録して」や「〇日の給料を計算して」といったリクエストを受けた場合、以下の手順で実行してください。

1. **情報の確認**:
   必要な情報（勤務日、勤務時間、時給、交通費）が揃っているか確認します。不足していればユーザーに質問してください。
   - `date`: 勤務日 (YYYY-MM-DD 形式)。ユーザーが「今日」と言った場合は本日の日付を計算して入れてください。
   - `hours`: 勤務時間 (数値形式。例: 5時間30分なら 5.5。20分など端数がある場合は 5.333 といった小数をそのまま渡してください。スクリプト内部で自動的に30分刻み(0.5単位)で切り捨てて計算されます。)
   - `wage`: 時給 (数値、円、デフォルトは1150円)
   - `transport`: 交通費 (数値、円。指定が無い場合はデフォルトで0として扱います)

2. **依存関係のインストール (初回のみ/エラー発生時)**:
   もし実行時に `openpyxl` ライブラリが見つからないなどのエラーが出た場合は、このスキルのディレクトリにある `requirements.txt` をインストールします。
   ```bash
   pip install -r c:/Users/Tetsu/OneDrive/バイト給料/.agents/skills/salary_calculator/requirements.txt
   ```

3. **スクリプトの実行**:
   `run_command` ツールを使って以下のPythonスクリプトを実行し、Excelに保存します。デフォルトの保存先は `c:/Users/Tetsu/OneDrive/バイト給料/` 配下の `YYYY年_バイト給料記録.xlsx` となります。
   ```bash
   python c:/Users/Tetsu/OneDrive/バイト給料/.agents/skills/salary_calculator/scripts/record_salary.py --date "日付" --hours <時間> --wage <時給> --transport <交通費>
   ```

4. **結果の報告**:
   スクリプトの出力（JSON形式）を受け取り、計算された日給と、Excelに記録が完了したことをユーザーにわかりやすく（専門用語を避けて）報告してください。Excelファイルが新たに作成・更新されたことも伝えます。
