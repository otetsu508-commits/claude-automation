# PDF要約スキル (pdf-summarizer)

このスキルは、PDFファイルを読み込み、その内容を日本語でわかりやすく要約して、指定したフォルダに保存します。

## 使用方法

Claude Codeで以下のように指示します：

**フォルダを指定する場合：**

```
このPDFを要約して c:\Users\Tetsu\OneDrive\大学授業\情報リテラシー1 に保存
```

**フォルダを指定しない場合（自動判定）：**

```
このPDFを要約して
```

または

```
講義資料を要約してまとめて
```

## 機能

- PDFファイルの内容を読み込み
- **画像ベースPDF（スキャン済み）のOCR処理に対応**
- 内容を日本語でわかりやすく要約
- 元のファイル名からわかりやすい要約ファイル名を生成
- 指定したフォルダ（または自動判定したフォルダ）に要約ファイルを保存

## OCR処理について

画像ベースのPDF（スキャン済みの講義資料など）も要約できます！

### OCR処理の流れ

1. まずReadツールでPDFを読み込みを試みます
2. 画像ベースで読み込めない場合、自動的にOCR処理を実行します
3. OCR処理で抽出されたテキストから要約を作成します

### 必要な環境

OCR処理には以下の環境が必要です（すでに設定済み）：

- Tesseract OCR（OCRエンジン）
- Pythonライブラリ：pytesseract, PyMuPDF, Pillow
- 日本語OCRデータ

### OCR出力ファイル

OCR処理の結果は以下の場所に保存されます：

```
C:\Users\Tetsu\OneDrive\大学授業\[教科名]\ocr_output.txt
```

## 出力例

元のファイル: `経営学概論　(4.社会の中の企業).pdf`

要約ファイル: `経営学概論_4_社会の中の企業_要約.md`

```markdown
# 経営学概論 - 要約

## 概要
企業と社会の関係性についての講義資料

## 主要ポイント
- 企業の社会的責任
- ステークホルダー理論
- 企業統治

## 詳細内容
### 企業の社会的責任
企業は利益追求だけでなく、社会に対する責任も果たす必要がある

### ステークホルダー理論
株主だけでなく、従業員、顧客、地域社会など全てのステークホルダーを考慮する

## 重要キーワード
- CSR
- ステークホルダー
- 企業統治
- サステナビリティ

---
元のファイル: 経営学概論　(4.社会の中の企業).pdf
作成日: 2026-05-19 15:00
```

## 対象フォルダ

デフォルトで以下の教科フォルダを対象とします：

- `大学授業/データリテラシー/`
- `大学授業/情報リテラシー１/`
- `大学授業/経営学概論/`
- （その他追加された教科フォルダ）

## 注意事項

- 長いPDFの場合は、ページ範囲を指定して要約を作成します
- 要約は簡潔かつわかりやすく記述されます
- 専門用語には説明が付きます
