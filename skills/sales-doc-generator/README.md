# 営業資料生成スキル クイックスタート

## このスキルでできること

- 提案書の自動生成
- 初回コンタクトメールの作成
- フォローアップメールの作成
- 導入事例の作成

---

## 5分で始める

### ステップ1：顧客情報を準備

以下の情報をメモしてください：

| 項目 | 例 |
|------|------|
| 顧客名 | 株式会社〇〇 |
| 業界 | 製造業 |
| 規模 | 従業員50名 |
| 課題 | 月次報告書作成に3日かかっている |
| 資料種類 | 提案書 / メール / フォロー |

### ステップ2：Claude Codeで実行

```
/skill sales-doc-generator
```

その後、以下の形式で入力：

```
顧客名: 株式会社〇〇
業界: 製造業
規模: 50名
課題: 月次報告書作成に3日かかっている
資料種類: 提案書
```

### ステップ3：出力確認

生成された資料は以下に保存されます：

- 提案書 → `06_marketing/proposals/YYYY-MM-DD_企業名/proposal.md`
- メール → `06_marketing/email-templates/YYYY-MM-DD_用途.md`

---

## 資料種類別の使い分け

| 資料 | いつ使う？ |
|------|-----------|
| **提案書** | 初回訪問後、正式提案時 |
| **初回メール** | 初めてDM/メール送信時 |
| **フォローメール** | 診断後・提案後のフォロー |
| **導入事例** | 成功事例を発信したい時 |

---

## テンプレートの直接使用

スキルを使わずにテンプレート直接編集も可能です：

| テンプレート | 場所 |
|-------------|------|
| 提案書 | `06_marketing/proposals/template.md` |
| 初回メール | `06_marketing/email-templates/template-first-contact.md` |
| フォローメール | `06_marketing/email-templates/template-follow-up.md` |
| 導入事例 | `06_marketing/case-studies/template.md` |

## 品質チェックリスト

資料送付前に確認してください：

- [ ] 顧客名・業界が正確か
- [ ] 具体的な作業例が入っているか
- [ ] 「ですます」調になっているか
- [ ] CTAが1つだけか
- [ ] 過度な保証をしていないか

## トラブルシューティング

**Q: 出力ファイルが見つかりません**
A: `06_marketing/` フォルダ以下を検索してください

**Q: 内容を修正したいです**
A: 出力されたファイルを直接編集するか、再度スキルを実行してください

**Q: 業界に合わせた具体例が欲しいです**
A: 課題入力時に「製造業の現場での報告書作成」のように具体的に記載してください

---

## 5体パイプラインとの連携

より詳細な資料を作成する場合は、グローバル設定の5体パイプラインを利用できます：

```
Strategist → 提案の切り口決定
Researcher → 顧客業界リサーチ
Writer → 資料作成
Editor → 簡潔化・修正
Publisher → 最終調整
```

詳細は `CLAUDE.md` を参照してください。
