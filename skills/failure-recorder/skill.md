# Daily Diary Recorder (日々の振り返り記録スキル)

## Overview
このスキルは、ユーザーの今日できたこと、失敗したこと、そこから学んだことをExcelの記録表に追記していくためのスキルです。
備え付けのスクリプト `scripts/record_diary.py` を実行して記録を行います。

## 使い方 (Workflow)

ユーザーから「振り返りを記録したい」「日記をつけて」という旨の依頼を受けたら、以下の手順で記録を行ってください。

1. **必要な情報を集める**
   記録には以下の3つの情報が必要です。ユーザーの入力から抽出するか、不足している場合は質問して確認します。
   - `achievement` (今日できたこと) ※任意、指定がない場合は「特になし」となります
   - `failure` (今日失敗したこと) ※任意、指定がない場合は「特になし」となります
   - `learning` (そこから学んだこと) ※任意、指定がない場合は「特になし」となります

2. **スクリプトの実行**
   情報が整理できたら、`scripts/record_diary.py` を実行して記録します。ダブルクォーテーションで囲むことを忘れないでください。

   ```bash
   python scripts/record_diary.py --achievement "〇〇が予定通りできた" --failure "△△でミスをした" --learning "××すべきだと学んだ"
   ```

3. **結果の報告**
   スクリプトの実行が成功し「振り返りの記録が完了しました！」と表示されたら、ユーザーに記録が完了したことを報告してください。

## スクリプトの仕様

`scripts/record_diary.py` は、内部で `C:\Users\Tetsu\.claude\日記保存エクセル\日々の振り返り記録.xlsx` という特定のExcelファイルに追記を行います。
- 初回アクセスなどでファイルやフォルダが存在しない場合は、スクリプトが自動で必要なフォルダおよび見出し付きのExcelファイルを作成します。
