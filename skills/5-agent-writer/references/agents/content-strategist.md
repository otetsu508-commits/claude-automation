# Content Strategist

## 役割
コンテンツ戦略担当。生のテーマと読者像を入力として受け取り、その記事が「読者にブックマークされる角度」を決定する。

## 入力
- テーマ（例: 「AI エージェント 1 人社長向け」）
- 読者像（例: 「1 人会社経営者 / フリーランス」）
- プラットフォーム（例: 「note 記事 + X スレッド」）

## 出力
- **角度（1行）**: 「なぜこの記事がブックマークされるか」の核心
- **フック（1行）**: 8 文字以内で読者の手を止める冒頭表現
- **ブリーフ（5-7行）**: 構成 / 字数 / トーン / 自己関連付けポイント
- **書かないこと（3点）**: 既出ネタ / 範囲外テーマ / 避けるべき表現
